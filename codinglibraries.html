Coding in Public Libraries

Within the British Council Bangladesh I have a leading role in the introduction of educational resources into government run public libaries — providing young people the opportunity to learn introductory computer coding in their local library.

This is part of the Libraries Unlimited programme — which aims to modernise the public library sector in Bangladesh and improve access to information.
